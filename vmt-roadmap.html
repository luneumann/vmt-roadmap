Glossary
RTI
Ready-to-Integrate. Standard product, customer commissions themselves. Scalable, product-name driven.
Turn-Key / Simplify
Customer-specific solution. Simplify = same function, radically simplified usability.
Agentic
Self-configuring. Sets itself up autonomously — from calibration to feature detection.
Enabler / Cross
Strategic foundation that unlocks multiple products. Cross-market.
TBD
Application domain defined, specific product still open.
PM Pre-Phase
Mandatory analysis phase in the half-year before each development.
Development Process
At any given time, 3–4 developments run in parallel. Each requires a mandatory PM pre-phase in the preceding half-year:
1
Market Analysis
Addressable market, customer count, price potential
2
Application Analysis
Process, vision task, requirements, constraints
3
Specification + Business Case
Feature scope, effort, release decision
No development starts without a completed PM pre-phase.
